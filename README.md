# Marie Curie Tribute Website
# Overview
This is an elegant, single-page tribute website dedicated to Marie Curie, the pioneering physicist and chemist. Built with pure HTML and CSS, it features 8 unique heading styles, a comprehensive timeline, achievements grid, and vintage-inspired design with ornamental elements and responsive layout.
​

# Features
8 Distinct Heading Styles: Classic line-through, ornamental divider, bordered box, underline accent, ribbon, corner brackets, side lines.

Interactive Timeline: 9 key life events (1867-1934) with custom markers and content cards.

Achievements Grid: 4 hover-animated cards highlighting Nobel Prizes, element discoveries, academic firsts, medical innovations.

Facts Section: 6 interesting facts in a 3-column responsive grid with lift-on-hover effects.

Vintage Typography: Times New Roman font, classic color palette (navy, cream, red accents), decorative shadows and borders.

Fully Responsive: Mobile-optimized with breakpoints at 968px and 600px.

# Usage
Save as task2.html and open in any modern browser.

Replace curie-image.jpg placeholder with actual portrait.

Customize colors (c0392b red, 34495e navy) or add images for enhanced visuals.

No JavaScript required—pure CSS animations and hover states.
​

# Technologies
HTML5: Semantic sections, timeline structure, grid layouts.

CSS3: Advanced pseudo-elements (::before/::after), CSS Grid/Flexbox, transforms, transitions, box-shadows, linear gradients, responsive media queries.
